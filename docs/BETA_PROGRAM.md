# Beta Program

## Goal

Onboard **10 merchants** to validate PulseOps SP Core setup, narrative flow, and perceived value before Theme Store submission.

## Target categories (public demo presets)

| Preset | Category |
|--------|----------|
| `supplement` | Supplement / Wellness |
| `skincare` | Skincare / Beauty |
| `creator` | Creator Brand |
| `electronics` | Electronics / Gadget |
| `lifestyle` | Apparel / Lifestyle |

Assign matching homepage and product templates from [DEMO_PRESETS.md](./DEMO_PRESETS.md).

Configure **hero-product navigation** (not Catalog-style menus). See [QUICK_START.md](./QUICK_START.md).

## Feedback objectives

1. **Setup friction** — time to first publishable product page
2. **Section usage** — which SP sections merchants keep, remove, or reorder
3. **Missing features** — gaps vs. Prestige / Impulse / Impact expectations
4. **Perceived value** — does the narrative stack feel premium and conversion-focused?
5. **Willingness to pay** — acceptable price point for a single-product OS theme
