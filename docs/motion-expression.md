---

## Purpose

Motion Expression translates the frozen Motion Foundation into chapter-level temporal identities.

The goal is not animated variety. It is temporal coherence through narrative difference. Each homepage chapter should communicate change differently because each chapter asks the visitor to understand a different kind of moment.

This document defines temporal expression only. It does not define animation timing, durations, easing, CSS, JavaScript, Liquid, component animations, scroll animations, microinteractions, engineering, or implementation.

---

## Expression Principle

Every chapter receives one dominant temporal relationship.

The relationship may be immediate orientation, gentle arrival, patient inspection, social continuity, reflective pause, controlled verification, human reassurance, efficient resolution, still commitment, or quiet continuation. If motion does not clarify the chapter's temporal role, it does not belong in that chapter.

Motion should follow the frozen buyer journey:

```
Arrival
  ->
Orientation
  ->
Inspection
  ->
Validation
  ->
Reflection
  ->
Verification
  ->
Reassurance
  ->
Resolution
  ->
Commitment
  ->
Completion
```

## 1. Hero

### Temporal Purpose

Hero motion removes first hesitation.

The opening should become understandable immediately. Motion should help the visitor recognize product, promise, and available path without asking for attention.

### Motion Character

Immediate, restrained, confident.

### Continuity Philosophy

Continuity should protect first orientation. Nothing should feel like it arrives from outside the story; the opening should feel already composed.

### Tempo Philosophy

Tempo should feel decisive, not expressive. Confidence matters more than temporal flourish.

### Stillness Philosophy

Stillness protects recognition. The Hero should not move in ways that delay understanding of product or promise.

### Emotional Pacing

The chapter should settle the visitor before the page begins to unfold.

### Relationship With Interaction

Interaction owns the first available actions. Motion may clarify readiness, but it must not create urgency or new behavior.

### Relationship With Photography

Photography establishes product presence. Motion should support recognition without disturbing the image's authority.

## Motion Progression

Motion evolves with the frozen buyer journey:

```
Arrival
  ->
Orientation
  ->
Inspection
  ->
Validation
  ->
Reflection
  ->
Verification
  ->
Reassurance
  ->
Resolution
  ->
Commitment
  ->
Completion
```

The page begins by orienting quickly, then widens into gentle continuity, slows for inspection, reassures socially, pauses for reflection, sharpens for verification, warms into identification, accelerates for resolution, becomes still for commitment, and closes with quiet continuation.

This progression prevents temporal repetition. Hero should not move like FAQ. Ingredients should not move like Formulation Philosophy. Scientific Confidence should not move like Human Proof. Future Self earns maximum stillness because confidence has already been built.

## Temporal Energy Progression

Temporal intensity should rise only when it clarifies change and fall as confidence grows:

```
Hero
  Immediate orientation

Editorial Outcomes
  Gentle arrival

Ingredients
  Deliberate inspection

Community Confidence
  Natural reassurance

Formulation Philosophy
  Long pause

Scientific Confidence
  Controlled precision

Human Proof
  Warm continuity

FAQ
  Efficient resolution

Future Self
  Maximum stillness

Footer
  Calm completion
```

Temporal energy is not visual activity. A chapter may feel calm while still communicating change clearly. The highest temporal energy belongs to understanding, inspection, verification, or resolution. The lowest belongs to reflection, commitment, and closure.

## Motion Consistency Rules

1. Movement follows behavior.
2. Motion protects reading.
3. Stillness grows as confidence grows.
4. Motion never competes with Photography.
5. Motion never replaces Interaction.
6. Movement should always clarify change.
7. Motion should preserve chapter identity.
8. Tempo should match confidence.
9. Continuity should preserve context.
10. Utility chapters should resolve efficiently.
11. Reflective chapters should protect pause.
12. Evidence chapters should avoid theatrical expression.
13. Human chapters should protect dignity.
14. Closing chapters should reduce temporal pressure.
15. If motion does not improve understanding, it should disappear.
